About Kung Fury's Task List

    Location:

  2. About

This site lists the tasks for the Kung Fury team of freedom fighters. The site is designed to react to user clicks on each row that represents a task on the home page. This site is also responsive to screensize and will automatically collapse the navigation bar when the screen width is less than 768px. A breadcrumb is also shown to help the user know where they are within the site.
